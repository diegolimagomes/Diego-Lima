Meus Projetos

Previsão do Tempo

Este projeto é um aplicativo de previsão do tempo construído com React.js, que permite aos usuários visualizar a previsão do tempo atual e para os próximos cinco dias para qualquer cidade do mundo.

Gerador de Senhas

Este projeto é uma aplicação React criada com Vite para gerar senhas aleatórias. A senha gerada é exibida na tela, e você pode copiá-la para a área de transferência com um clique em um botão.

Formulário

Este projeto consiste em um formulário interativo, estilizado com Bootstrap, para coletar informações pessoais, preferências de gênero, treinamentos realizados e qualidades do usuário, com foco em uma interface responsiva e amigável.

Calculadora

A criação de uma calculadora é um projeto clássico e essencial para quem está aprendendo desenvolvimento de software, pois envolve a aplicação de diversas funcionalidades fundamentais, como manipulação de eventos, cálculos lógicos e interface de usuário.
